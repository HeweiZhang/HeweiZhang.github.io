二、RecyclerView中使用EditText

网上看到博友在ListView中使用EditText时，点击editText后软键盘弹出后会导致光标失去焦点，这里我在使用RecyclerView的时候并没有发现光标问题，这里先跳过光标问题。

使用EditText是主要问题在于如何获取到EditText的值，以及刷新adapter时，出现editText值错乱
